### How To Use

Grammatical hints appear when you enter the keyword of the component name, and you can quickly complete it after selecting it.

<img src="https://gitee.com/forestxieCode/resource/raw/master/naive-ui/01.gif" style="width: 100%; display: block; border-radius: 4px; margin-top: 16px">

Moving the mouse to the component name will display the document address of the component, which can be clicked to jump.

<img src="https://gitee.com/forestxieCode/resource/raw/master/naive-ui/02.png" style="width: 100%; display: block; border-radius: 4px; margin-top: 16px">

Press space or attribute keywords in the attribute input range of the label to display attribute completion prompts and attribute information.

<img src="https://gitee.com/forestxieCode/resource/raw/master/naive-ui/03.png" style="width: 100%; display: block; border-radius: 4px; margin-top: 16px">
